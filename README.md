# Is Blue Side Actually Better? Examining Regional Playstyle Differences and Side Selection Advantage in Professional League of Legends

**By:** Loren Pan

## Introduction
This project uses a cleaned subset of the 2025 League of Legends esports match data from Oracle's Elixir. After filtering the dataset to the four major leagues (`LTA N`, `LEC`, `LPL`, and `LCK`) alongside other data cleaning, the resulting working dataframe contains 3758 rows with 157 columns. Of those 157 columns, the ones we used in this study are:

- **`league`**:
- gamelength
- goldat15
- firstbaron
- result
- league
- opp_deathsat15
- split
- firstPick
- killsat15
- golddiffat15
- wpm
- ckpm
- firstdragon
- firsttower
- vspm
